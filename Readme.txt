The code was developed in Microsoft Visual Studio 2015 Community Edition. 

Following instructions need to be followed for running the program successfully.

1) The project has 3 files.
	1: main.cpp
	2: A header file animal.h
	3: animal.cpp

2) On executing the code the display window will show 2 animals and a soiled path and a grassy landscape

3) The user can press 'r' and 't' to rotate the camera, to get a view from different angles.

4) The user can press 'b' and 'f' to move the animal backward and forward respectively.

5) The user can press 'w' to toggle btw Solid and Wired mode only the animals in the scene will be wired.
